# dbms-digest

Personal weekly digest on what's going on in the DBMS world.

Each digest is a terse, ad-free, fact-checked roundup of PostgreSQL and wider DBMS news,
research, and engineering for one **Monday → Sunday** week. Files live in [`digests/`](digests/),
named by the Monday that starts the covered week. New digests are generated by the
`weekly-dbms-digest` skill; see [`CLAUDE.md`](CLAUDE.md) for the auto-run routine.
